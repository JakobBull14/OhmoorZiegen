🔒 Datenschutz

Informationen zur Verarbeitung von Daten in der Schulziegen-App.

1. Worum geht es auf dieser Seite?

Diese App stellt die Schulziegen vor und bietet interaktive Funktionen wie Voting, Memory, Quiz und Ranglisten.

2. Welche Daten werden genutzt?

In der App können folgende Daten verarbeitet werden:

  • ein selbst gewählter Spielername für Ranglisten und Spiele
  • Spielergebnisse aus Memory und Quiz
  • abgegebene Stimmen beim Voting
  • technische Informationen, die nötig sind, damit die App funktioniert

3. Lokale Speicherung im Browser

Die App nutzt den lokalen Speicher deines Browsers (localStorage). Dort werden zum Beispiel gespeichert:

  • dein Spielername
  • ob du deinen Namen bereits bestätigt hast
  • ein technischer Schlüssel für Abstimmungen
  • welche Ziege du gewählt hast
  • ob ein Admin eingeloggt ist

Diese Informationen werden auf deinem Gerät gespeichert und helfen dabei, dass die App für dich richtig funktioniert.

4. Ranglisten

Wenn du Memory oder Quiz spielst, können dein Spielername und dein Ergebnis in einer Rangliste gespeichert werden. Dazu gehören zum Beispiel:

  • dein Spielername
  • dein Ergebnis
  • die Anzahl der Punkte oder Züge
  • der Zeitpunkt des Eintrags

Diese Daten werden nur dafür genutzt, die Ranglisten der App anzuzeigen.

5. Voting

Beim Abstimmen wird ein technischer, anonymer Schlüssel verwendet. Dadurch kann die App erkennen, ob auf einem Gerät bereits abgestimmt wurde oder eine Stimme geändert wird. Ein direkter Rückschluss auf eine Person ist darüber nicht vorgesehen.

6. Zweck der Datenverarbeitung

Die Daten werden nur verarbeitet, damit die Funktionen der App möglich sind, insbesondere:

  • Darstellung der Ziegen
  • Durchführung von Voting, Memory und Quiz
  • Anzeige von Ranglisten
  • Verwaltung der Inhalte im Adminbereich

7. Löschung von Daten

Lokale Daten im Browser können gelöscht werden, indem die Browserdaten gelöscht werden. Online gespeicherte Ranglisten- oder Voting-Daten können im Adminbereich der App verwaltet oder gelöscht werden.
